# hyperbola
Программа строит графики преобразования фурье (FFT) или спектральную плотность мощности (PSD)

Формат данных предполагает, что данные разделены через "," и располагаются в таком формате: время, величина_1 , величина_2 , величина_3.
В случае необходимости, разделитель можно изменить на другой и считывать данные с другой колонки.
Для примера в source лежит пример данных "example_data.csv"
